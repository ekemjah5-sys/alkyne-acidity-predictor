# Alkyne Acidity Predictor

A rapid, physical-organic informed machine learning tool for predicting terminal alkyne pKa values using semi-empirical quantum descriptors. This platform achieves near-DFT fidelity in milliseconds, bypassing traditional computational bottlenecks to accelerate high-throughput drug design and catalyst screening workflows.

## 📊 Core Performance Metrics
* **Accuracy (R²):** 0.8611
* **Error Boundary (RMSE):** 1.17 pKa units
* **Prediction Latency:** < 3.0 seconds per candidate molecule
